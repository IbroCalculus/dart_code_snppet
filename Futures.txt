
================== Definitions of Future, Async, Await ===================

- Future
A Future represents a value that will exist later. Think of it like a promise of future data
In other words, A Future<T> means: "I do not have the value now, but I will give you a value of type T later."
Future<String>: means "A String will arrive later."

- Async
Async means: "This function works with Futures and may pause using await."
It mainly enables await inside the function.

Important Misconception
  async itself does NOT create delays.  async does NOT make code asynchronous by magic. The Future does that.

- Await
await means "Pause THIS async function until the Future completes."
E.g: String value = await getData(); Meaning: "Wait here until getData() finishes."

NB: await pauses ONLY the current async function. NOT the whole app. Flutter UI keeps running.

- .then()
.then() means: "When this Future finishes, run this callback."



======================== Examples =======================

---------------- Using Await ------------

void main() async {

  print("Start");
  String value = await getData();
  print("Future returned value: ${value}");
  print("End");
}

Future<String> getData() {
  return Future.delayed(Duration(seconds: 5), () {
    return "Done after 2 seconds";
  },);
}


OUTPUT:
  Start
  Future returned value: Done after 2 seconds
  End


------------------ Using .then ---------------

void main() {

  print("Start");

  getData().then((value) {
    print("Future returned value: ${value}");
  }).catchError((error) {
      print("Error: ${error}");
    },
  );

  print("End");
}

Future<String> getData() {
  return Future.delayed(Duration(seconds: 5), () {
    return "Done after 2 seconds";
  },);
}


OUTPUT:
  Start
  End
  Future returned value: Done after 2 seconds

Future<Map<String, dynamic>> fetchData() async {
  final response = await http.get(Uri.parse("https://jsonplaceholder.typicode.com/posts/1"));
  if (response.statusCode == 200) {

    final String jsonStringResponse = response.body;			// response.body returns the API response as a String.
    final Map<String, dynamic> jsonResponse = json.decode(jsonStringResponse);    // json.decode() converts the JSON String into a Dart object such as a Map<String, dynamic> or List<dynamic>.
    var postTitle = jsonResponse["title"];			// Assuming it's a json data response with a key of "title"

    // print("JSON STRING: $jsonStringResponse");
    // print("JSON RESPONSE: $jsonResponse");
    // print("POST TITLE: $postTitle");

    return jsonResponse;

  } else {
    // If the server did not return a 200 OK response, throw an exception.
    throw Exception('Failed to load data from the API');
  }
}
